ПОШАГОВОЕ РУКОВОДСТВО
======================
1) Заведи аккаунты: GitHub и Streamlit (streamlit.io).
2) Создай на GitHub новый репозиторий 'waybill-maker'.
3) Залей туда файлы из этого архива: app.py, requirements.txt, utils.py, supplier_profiles.yaml, sample_template.xlsx.
4) На Streamlit Cloud → New app → выбери репозиторий, укажи app.py и нажми Deploy.
5) Через минуту сайт готов: загружай PDF-счета и качай готовые waybill.xlsx.
